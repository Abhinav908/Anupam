# Attendance System Documentation

## Features  
- Track attendance for students and staff.  
- Admin panel for managing user accounts.  
- Generate attendance reports.  
- Notifications for absences and late arrivals.  

## How to Use  
### For Students  
1. Log in to the attendance system using your student credentials.  
2. Check your attendance record.  
3. Request approval for any discrepancies in attendance.  

### For Admins  
1. Log in to the admin panel.  
2. Manage student accounts and attendance records.  
3. Generate and download reports.

## Setup Instructions  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/Abhinav908/Anupam.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd Anupam  
   ```  
3. Install the required dependencies:  
   ```bash  
   npm install  
   ```  
4. Start the application:  
   ```bash  
   npm start  
   ```
